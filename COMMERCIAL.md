# BTreePlus – Commercial Edition

The **Enterprise Edition** unlocks advanced operations for production systems:

## Features

- **Erase** — safe structural delete
- **Range Scans** — high-throughput ordered traversal
- **Cursors** — `Seek(key)` / `Next()` for efficient iteration
- **Maintenance Tools** — integrity checks, compaction helpers, diagnostics

## Distribution

- Delivered as **binary** via a **private NuGet feed**
- Enabled at build/runtime via a **signed license token**
- Source code is not distributed

## Build / License

- Public builds expose only the Community API.
- Enterprise builds are compile-gated (symbol) and perform **runtime license verification**.
