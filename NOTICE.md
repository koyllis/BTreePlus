# NOTICE

This repository contains **documentation and example code** for using **BTreePlus**.

- The **BTreePlus engine** is distributed **as a binary via NuGet**.
- **Engine source code is not included** in this repository.
- The **Community Edition** exposes the public API documented in `API.md`.
- **Enterprise features** (Erase, range scans, cursors, maintenance) are available to licensed customers via a **private NuGet feed** and runtime **license token**. See `COMMERCIAL.md` for details.
